PHP (Hypertext Preprocessor) je strežniški skriptni jezik, zasnovan za razvoj dinamičnih spletnih strani. Uporablja se za obdelavo podatkov, povezovanje z bazami in ustvarjanje vsebin, prilagojenih uporabnikom.



## Osnovna uporaba PHP-ja

PHP kodo lahko vstavimo znotraj HTML dokumenta z oznakami `<?php` in `?>`. Koda se izvaja na strežniku, rezultat pa se pošlje brskalniku kot HTML.

### Primer osnovnega PHP-ja:

```
<?phpecho "Pozdravljen, svet!";
?>
```

Rezultat:

```
Pozdravljen, svet!

```



## Spremenljivke v PHP-ju

Spremenljivke v PHP-ju se začnejo z `$` in lahko shranjujejo različne tipe podatkov, kot so besedila, številke in tabele.

### Primer:

```
<?php$ime = "Janez";
$starost = 25;

echo "Ime: " . $ime . ", Starost: " . $starost;
?>
```



## Pogojni stavki

PHP omogoča uporabo pogojnih stavkov za izvajanje določenega dela kode glede na pogoje.

### Primer:

```
<?php$starost = 20;

if ($starost >= 18) {
    echo "Polnoleten si.";
} else {
    echo "Nisi polnoleten.";
}
?>
```



## Zanke v PHP-ju

Zanke omogočajo večkratno izvajanje dela kode.

### Primer for zanke:

```
<?phpfor ($i = 1; $i <= 5; $i++) {
    echo "Številka: $i<br>";
}
?>
```

### Primer while zanke:

```
<?php$i = 1;
while ($i <= 5) {
    echo "Številka: $i<br>";
    $i++;
}
?>
```



## Funkcije

Funkcije so bloke kode, ki jih lahko pokličemo, kadar jih potrebujemo.

### Primer:

```
<?phpfunction pozdrav($ime) {
    return "Pozdravljen, $ime!";
}

echo pozdrav("Marija");
?>
```



## Delo z obrazci

PHP omogoča obdelavo podatkov iz obrazcev, kar je pogosto uporabljeno pri dinamičnih spletnih straneh.

### Primer HTML obrazca:

```
<form method="post" action="obdelava.php">
  Ime: <input type="text" name="ime"><input type="submit" value="Pošlji"></form>
```

### Obdelava podatkov v PHP:

```
<?phpif ($_SERVER["REQUEST_METHOD"] == "POST") {
    $ime = $_POST['ime'];
    echo "Pozdravljen, $ime!";
}
?>
```



## Povezava z bazo podatkov

PHP omogoča enostavno povezovanje z bazami podatkov, kot je MySQL.

### Primer:

```
phpKopiraj kodo
```



## Prednosti PHP-ja

*   Združljiv z večino spletnih strežnikov in baz podatkov.

*   Dinamičen in močan za razvoj spletnih aplikacij.
