HTML (HyperText Markup Language) je osnovni gradnik spleta, ki omogoča ustvarjanje strukturiranih dokumentov in spletnih strani. S pomočjo HTML-ja definiramo vsebino in njeno osnovno hierarhijo, ki jo brskalniki interpretirajo in prikazujejo uporabnikom.

## Osnovna struktura HTML-dokumenta

Vsak HTML-dokument se začne s strukturo, ki vključuje glavne oznake: `<html>`, `<head>`, in `<body>`.

### Primer osnovne strukture

```
htmlKopiraj kodo
```



## Naslovi

Naslovi so ključni za organizacijo vsebine. HTML podpira šest stopenj naslovov, od `<h1>` do `<h6>`, kjer `<h1>` označuje najpomembnejši naslov.

### Primeri naslovov:

*   `<h1>` Glavni naslov

*   `<h2>` Podnaslov

*   `<h3>` Manjši podnaslov

*   `<h4>` Podpodnaslov

*   `<h5>` Skoraj najmanjši podnaslov

*   `<h6>` Najmanj pomemben naslov



## Poudarki

Uporabite različne oznake za poudarjanje besedila:

*   **Krepko besedilo**: `<strong>` ali `<b>`

*   *Ležeče besedilo*: `<em>` ali `<i>`

*   Prečrtano besedilo: `<del>`



## Seznami

HTML omogoča urejene in neurejene sezname.

### Neurejen seznam:

```
htmlKopiraj kodo
```

### Urejen seznam:

```
htmlKopiraj kodo
```



## Koda

V HTML-ju lahko prikažemo kodo z oznako `<code>` za enovrstično kodo ali `<pre>` za blok kode.

### Primer:

**Inline koda**: `<div>To je div element.</div>`

**Blok kode**:

```
htmlKopiraj kodo
```
